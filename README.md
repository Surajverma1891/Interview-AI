# Interview AI - Smart Placement Preparation Tool 🤖

An AI-powered mock interview platform built with the **MERN Stack** and **Google Gemini API** to help students practice technical interviews with real-time feedback.

## 🚀 Features
- **AI-Powered Questions:** Generates dynamic, role-specific technical questions using Google Gemini.
- **Real-time Feedback:** Analyzes user responses and provides instant constructive suggestions.
- **Secure Authentication:** Integrated user login and profile management.
- **Responsive UI:** Modern, cinematic dark-themed interface built with Tailwind CSS.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Lucide React
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **AI Integration:** Google Gemini Pro API
- **Auth:** Clerk / JWT (jo bhi aap use kar rahe ho)
